# **App Name**: Siddhant Sahoo Portfolio

## Core Features:

- Dynamic Profile Content Management: Enable intuitive configuration of all profile textual and visual elements including portfolio title, tagline, introduction, skill highlights, work experience details, project information, testimonial quotes, social links, and parallax image sequences.
- Scroll-Driven Parallax Hero: Implement a full-screen WebP image sequence as the hero background, smoothly animating frames based on scroll position for a cinematic and responsive visual experience.
- Themable Dark Mode Interface: Provide a default dark mode with user-selectable accent colors (Deep Sky Blue or Vivid Cerulean) dynamically applied to buttons, highlight lines, section markers, and active UI states.
- Detailed Work Experience Section: Display professional work experience chronologically, segmented into distinct roles with comprehensive bullet-point descriptions for each achievement and responsibility.
- Interactive Projects Showcase: Present projects in an engaging grid layout, featuring thumbnails, concise 1-2 line descriptions, and clickable links to redirect users to detailed project pages or external resources.
- Elegant Fixed Navbar: Implement a fixed, top-positioned navigation bar with rounded corners, a subtle white outline, 70% opacity, and a blurring effect for overlapped content.
- AI Content Assistant: Offer a generative AI tool to assist in drafting, refining, or summarizing textual content for sections like project descriptions, about me statements, or even potential testimonial drafts, ensuring compelling and concise language.

## Style Guidelines:

- Overall Color Scheme: Predominantly dark with high-contrast, vibrant accents to create a premium, modern, and cinematic aesthetic. Dark mode is the exclusive operational mode.
- Primary Accent Color: A brilliant, electric blue (`#00B3FF`), chosen for its association with technology and data, to highlight interactive elements, key information, and create visual anchors.
- Background Color: A deep, sophisticated bluish-grey (`#1A2226`), providing a rich, dark canvas that allows vibrant accent colors to pop, and ensures comfortable readability.
- Secondary Accent Color: A refreshing turquoise/cyan (`#52F5DB`), used for complementary details, hover states, and to add a subtle layer of visual depth without competing with the primary accent.
- Headline Font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired character, delivering a bold and impactful statement in titles and major headings.
- Body Text Font: 'Inter' (sans-serif) for its excellent readability and neutral, professional appearance, ensuring clear communication of all portfolio details.
- Social Icons: Minimalist, monochrome vector icons for platforms like YouTube, LinkedIn, GitHub, and X, maintaining a clean, high-end design consistency.
- Single-Page Immersive Design: A continuous scrolling experience with distinct, well-defined sections, providing a fluid narrative flow through the portfolio.
- Hero Section Layout: Precision-designed with left-aligned identity block, horizontal skill highlights, right-aligned value proposition, and bottom-centered social icons, overlaying a parallax background with a rounded border bottom, precisely matching the visual composition.
- Fixed Navigation Bar Styling: Features rounded corners, a subtle white outline, and a frosted glass-like appearance achieved with 70% opacity and blur.
- Parallax Scroll Behavior: WebP image sequence frames advance/reverse in direct correlation with scroll position, providing a smooth and responsive cinematic animation.
- Pre-Load Experience: A full-screen loading overlay with a logo and horizontal progress bar ensures all parallax frames are loaded before revealing the hero section, guaranteeing smooth initial interaction.
- Dynamic Content Transitions: Smooth fade-in and fade-out animations are applied to textual and visual elements during theme changes or content updates, enhancing user experience.
